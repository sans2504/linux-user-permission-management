# Linux User and Permission Management

A comprehensive Red Hat Linux user and permission management system with automated scripts and documentation.

## Features
- User account management
- Group management
- File permission management
- User auditing and reporting
- Backup and restore capabilities

## Prerequisites
- Red Hat Linux or CentOS
- Root/sudo privileges
- Bash shell

## Quick Start
1. Clone this repository
2. Make scripts executable: `chmod +x scripts/*.sh`
3. Run desired scripts with sudo privileges

## Scripts Overview
- `user_management.sh` - Create, modify, and delete users
- `group_management.sh` - Manage groups and group memberships
- `permission_manager.sh` - Set and manage file permissions
- `audit_users.sh` - Audit user accounts and permissions
- `backup_users.sh` - Backup user configurations

## Documentation
Check the `docs/` directory for detailed guides and best practices.
